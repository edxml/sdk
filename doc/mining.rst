==============
Concept Mining
==============

The SDK contains a concept mining implementation that builds an in-memory graph from event data. The subject of EDXML concept mining is briefly touched in `Introducing EDXML <http://edxml.org/intro/>`_ and explained more extensively `here <http://edxml.org/concept-mining/>`_.
